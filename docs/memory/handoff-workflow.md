# Quy uoc dong bo: HANDOFF.md

Du an nay duoc code song song bang ca Claude Code va Codex (khi Claude Code bi
gioi han se chuyen sang dung Codex, hoac nguoc lai). De 2 cong cu AI khong lam
that lac ngu canh cua nhau hoac tao nhieu file trang thai chong cheo khi thay
phien lam viec, du an dung **mot file duy nhat** o goc repo: `HANDOFF.md`.

`HANDOFF.md` chua: muc tieu san pham, quyet dinh kien truc, rang buoc moi truong,
roadmap tom tat, trang thai phase hien tai, checklist viec can lam, bang ket qua
do dac (tu dong cap nhat boi script `summarize_results.py`), va nhat ky cap nhat.

## Quy tac lam viec

- LUON doc toan bo `HANDOFF.md` truoc khi bat dau bat ky task nao trong repo nay.
- Sau khi hoan thanh hoac thay doi gi quan trong, cap nhat lai `HANDOFF.md` (muc
  "Quyet dinh moi", "Van de dang mo", "Nhat ky cap nhat") thay vi tao file
  markdown trang thai moi.
- Cac file trong `docs/memory/` (thu muc nay) la ghi chu boi canh/ly do, cap nhat
  it thay doi hon HANDOFF.md - khong dung de theo doi tien do.
- File `CLAUDE.md` o goc repo duoc Claude Code tu dong nap moi phien lam viec, co
  nhiem vu duy nhat la tro toi `HANDOFF.md`.
